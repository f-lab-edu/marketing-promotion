# 마케팅 프로모션 플랫폼

## 1. 프로젝트 개요
- 고객에게 기업들의 마케팅 프로모션을 제공하는 플랫폼 입니다.
- 고객사는 플랫폼을 통해, 고객에게 제공할 프로모션을 등록할 수 있습니다.
- 고객은 제공된 프로모션을 참여 / 취소 / 완료할 수 있고, 프로모션을 완료하면 리워드를 지급받습니다.

## 2. Use Case

## 2-1. ADMIN

### 1) 고객사 관리
- 관리자는 `고객사 관리` 페이지에서 고객사를 등록하고 조회할 수 있습니다.
- `고객사 관리 > 고객사 등록` 페이지에서 서비스를 이용할 고객사를 등록합니다.
- 등록된 고객사 목록은 `고객사 관리`페이지에서 조회할 수 있습니다. 
- 조회된 고객사는 `고객사 상세`페이지를 통해 상세정보 조회 및 수정할 수 있습니다.

### 2) 프로모션 관리
- 관리자는 `프로모션 관리` 페이지에서 프로모션을 등록하고 조회합니다.
- `프로모션 관리 > 프로모션 등록`페이지에서 프로모션을 등록합니다.
  - 고객사를 선택해, 고객사 별로 프로모션을 등록합니다.
  - 프로모션에 참여정책을 추가할수 있습니다.
- 등록된 프로모션 목록은 `프로모션 관리`페이지에서 조회할 수 있습니다.
- 조회된 프로모션는 `프로모션 상세`페이지를 통해 상세정보 조회 및 수정할 수 있습니다.

### 3) 프로모션 참여이력 관리
- 관리자는 `프로모션 관리 > 프로모션 상세` 에서 `프로모션 참여이력` 페이지로 이동합니다.
- `프로모션 참여이력` 에는 일자별 누적 참여이력 통계와 일자별 고객 참여이력을 조회할 수 있습니다. 
  - 일자별 누적 참여이력 통계는 일자별 참여, 완료, 취소, 리워드금액을 조회할 수 있습니다.
  - 일자별 고객 참여이력은 일자별 참여한 고객의 참여이력 목록을 조회할 수 있습니다.
  - 통계정보와 참여이력 목록는 엑셀 다운로드 가능합니다.

## 2-2. USER

### 1) 회원가입
- 고객은 로그인 페이지에서 회원가입합니다.
- 회원가입은 통신사 본인인증을 통해, CI 당 한개의 계정만 가입할 수 있습니다.
- 이미 회원가입한 CI 의 고객인 경우, `이미 가입된 회원입니다.` 문구가 표기됩니다.
- 이미 회원가입된 ID 를 사용할 경우, `이미 사용 중인 ID 입니다.` 문구가 표기됩니다.

### 2) 로그인
- 회원가입한 고객은 로그인 페이지에서 로그인합니다.

### 3) 참여가능한 프로모션 조회
- 고객은 `프로모션`페이지에서 참여가능한 프로모션 목록을 조회할 수 있습니다. (최대 10개씩 페이징 처리)
- 프로모션은 리워드 금액이 큰 순서로 표기됩니다.
- 프로모션을 선택하면 `프로모션 상세` 페이지로 이동합니다.

### 3) 프로모션 참여
- `프로모션 상세` 페이지에서 참여버튼을 통해 참여 가능합니다.
  - 이미 참여한 프로모션에 경우, `이미 참여한 프로모션입니다.` 문구가 표기됩니다.
  - 참여 인원수를 초과한 경우, `제한됨 참가인원수가 마감되었습니다.` 문구가 표기됩니다.
  - 참여 가능한 기간이 넘어간 경우, `참여 가능한 기간이 아닙니다.` 문구가 표기됩니다.

### 4) 프로모션 참여이력 조회
- `프로모션 참여이력`페이지 에서 고객은 참여한 참여이력 목록을 조회할 수 있습니다.
- 참여이력을 선택하면 `프로모션 참여이력 상세` 페이지로 이동합니다.
- 프로모션 별로 상태가 표기됩니다.
  - 상태는 `진행중`, `참여취소`, `리워드 지급 대기`, `리워드 지급 완료` 가 있습니다.  

### 5) 프로모션 참여취소
- `프로모션 참여이력 상세`페이지에서 `참여취소` 버튼을 통해 참여취소할 수 있습니다.
- 참여취소가 완료되면 `프로모션 참여이력`페이지로 이동합니다.

### 5) 프로모션 완료
- 고객은 프로모션의 내용을 프로모션 컨텐츠 URL 링크에서 수행합니다.
- 제한 기간 내 프로모션 컨텐츠를 완료하면, 프로모션 완료 페이지로 이동합니다.
- 이후 `프로모션 참여이력 상세` 에는 `리워드 지급 대기` 상태를 거쳐 `리워드 지급 완료` 상태가 되며 리워드 지급이 완료됩니다.
